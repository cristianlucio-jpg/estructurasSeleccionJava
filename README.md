# Proyecto: estructurasSeleccionJava

## Descripción
Este proyecto contiene una serie de ejercicios en **Java** que aplican las **estructuras de selección** (`if`, `else`, `switch`) para resolver problemas básicos de lógica y programación.  
Forma parte de un portafolio académico y profesional orientado a la práctica de **estructuras de control**.

---

## Objetivos
- Comprender y aplicar condicionales simples, compuestos y múltiples.
- Utilizar operadores relacionales y lógicos (`>`, `<`, `==`, `!=`, `&&`, `||`).
- Desarrollar programas modulares y bien documentados con **código limpio**.
- Practicar entrada y salida de datos con `Scanner` y `System.out`.

---

## Contenido del proyecto

| Ejercicio          | Clase                  | Descripción |
|--------------------|------------------------|-------------|
| DiasSemana         | `DiasSemana.java`      | Muestra el día de la semana según número ingresado (1–7). |
| MayorDosNumeros    | `MayorDosNumeros.java` | Determina el mayor entre dos números enteros. |
| MayorTresNumeros   | `MayorTresNumeros.java`| Determina el mayor entre tres números usando condicionales anidados. |
| MayorCuatroNumeros | `MayorCuatroNumeros.java` | Determina el mayor entre cuatro números con condicionales múltiples. |
| SignoZodiaco       | `SignoZodiaco.java`    | Determina el signo zodiacal según fecha de nacimiento. |
| GradosEstados      | `GradosEstados.java`   | Determina el estado del agua (sólido, líquido, gaseoso) según temperatura. |
| BisiestoNoBisiesto | `BisiestoNoBisiesto.java` | Determina si un año es bisiesto o no. |

---

## Contenido temático
- **Selección simple** → `if`
- **Selección compuesta** → `if` + `else`
- **Selección múltiple** → `switch`
- **Condicionales anidados**
- **Operadores relacionales** → `>`, `<`, `<=`, `>=`, `==`, `!=`
- **Operadores lógicos** → `&&`, `||`
